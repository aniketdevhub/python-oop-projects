# 🧠 NLP Application using Python & NLP Cloud API

## 📌 Overview

This project is a menu-driven Natural Language Processing (NLP) application developed in Python using Object-Oriented Programming (OOP). It provides a simple authentication system and integrates the NLP Cloud API to perform real-world NLP tasks such as Named Entity Recognition (NER) and Sentiment Analysis.

The project was built as part of my journey to strengthen my understanding of Python, OOP concepts, API integration, and software development practices.

---

## ✨ Features

* 🔐 User Registration and Login System
* 👤 Encapsulated User Database using Python OOP
* 🧠 Named Entity Recognition (NER)
* 💬 Sentiment Analysis
* 🌐 NLP Cloud API Integration
* ⚠️ Exception Handling for API Requests
* 🔒 Environment Variable Support for API Keys (`.env`)

## 🚀 Installation

1. Clone the repository.

```bash
git clone https://github.com/aniketdevhub/python-oop-projects.git
```

2. Navigate to the project directory.

```bash
cd NLP-App
```

3. Install the required packages.

```bash
pip install -r requirements.txt
```

4. Create a `.env` file in the project directory.

```env
NLP_API_KEY=YOUR_API_KEY
```

5. Run the application.

```bash
python NLPapp.py
```

---

## 📖 How It Works

1. Register a new account or log in with an existing account.
2. Choose one of the available NLP features.
3. Enter the required text.
4. The application sends the request to the NLP Cloud API.
5. Results are displayed in the console.
